```markdown
# Interactive Accounting Demo

نسخة مبسطة من نظام محاسبي تفاعلي مبني بـ Node.js + Express + SQLite.

المميزات:
- تسجيل قيود يومية مع تحقق توازن المدين/الدائن.
- قائمة الأستاذ المجمعة (trial balance).
- نظام مخزون مبسط (perpetual): تسجيل مشتريات ومبيعات وتحديث الكمية ومتوسط التكلفة.
- جدول إهلاك خطّي قابل للحساب.
- واجهة أمامية تفاعلية تعتمد على Chart.js وواجهة المستخدم التي قدمتها مع تطويرات.

التشغيل محلياً:
1. تثبيت الحزم:
   npm install

2. تشغيل السيرفر:
   npm start

الواجهة متاحة على:
http://localhost:3000

ملاحظات:
- هذه نسخة مبدئية قابلة للتطوير. يمكن إضافة مصادقة، حماية، تسجيل نشاط، تقارير مطبوعة، استيراد/تصدير، دعم عدة عملات، وواجهات مستخدم أفضل.
```
